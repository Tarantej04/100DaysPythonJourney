🕹️ Day 8 – Caesar Cipher Encryption Tool 🔐

Overview

A simple implementation of the classic Caesar Cipher encryption technique using Python!
Users can choose to encode or decode their messages by shifting letters forward or backward through the alphabet. Non-alphabet characters are preserved so punctuation and spaces remain intact.

✨ Features

Supports both encryption (encode) and decryption (decode)

Handles custom shift values

Preserves spaces and punctuation

Wraps around the alphabet using modular arithmetic

Allows repeated runs without restarting the program

Displays clean result output for each operation

⚙️ How to Run

Clone this repository or copy the project files.

Ensure the following file is in the same folder:

caesarCipher_art.py   # Contains the logo ASCII art


Run the script with:

python caesarCipher.py

ASCII Art for visual banner

Modular arithmetic for shifting logic
