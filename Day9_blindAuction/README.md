🕹️ Day 9 – Silent Auction Bidding Game 💰

Overview

A fun interactive silent auction simulation built in Python! Players secretly enter their names and bids, and once bidding ends, the program determines and announces the highest bidder. The console clears between bidders to keep bids hidden, creating a realistic “silent auction” feel.

✨ Features

Accepts multiple bidders with unique bid amounts

Stores all bids using a Python dictionary

Automatically determines the highest bidder

Clears screen between turns to protect bid privacy

Provides clean winner output with highest bid amount

⚙️ How to Run

Clone this repository or copy the project files.

Ensure the following file is in the same folder:

art.py   # Contains the logo ASCII art


Run the script using:

python blindAuction.py


🧠 Technologies Used

Python 3

Dictionaries for data storage

ASCII art for visual banner

Input/Output for interactive gameplay
