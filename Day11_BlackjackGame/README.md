# 🃏 Blackjack Game ♠️♥️♣️♦️

## Overview

A command-line implementation of the classic Blackjack card game built in Python. Players compete against a computer dealer, draw cards strategically, and try to get as close to 21 as possible without going over. The game includes Blackjack detection, Ace value adjustment, and automated dealer behavior following standard Blackjack rules.

## ✨ Features

* Deals random cards to both player and dealer
* Detects Blackjack automatically
* Adjusts Ace values from 11 to 1 when necessary
* Allows players to hit or stand
* Implements dealer drawing logic until reaching 17
* Determines the winner based on Blackjack rules
* Supports multiple rounds of gameplay
* Displays final hands and scores at the end of each game

## ⚙️ How to Run

Clone this repository or copy the project files.

Ensure the following file is in the same folder:

```text
art.py   # Contains the Blackjack ASCII art logo
```

Run the script using:

```bash
python BlackjackGame.py
```

## 🧠 Technologies Used

* Python 3
* Lists for storing player and dealer cards
* Functions for game logic and score calculation
* Random module for card dealing
* Conditional statements for game rules
* Loops for gameplay control
* ASCII art for visual banner
