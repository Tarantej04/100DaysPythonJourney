# 🧮 The Calculator Project ➕➖✖️➗

## Overview

A simple interactive calculator built in Python that performs basic arithmetic operations. Users can continue calculations using previous results, start a new calculation at any time, or exit the program. The project uses functions and a dictionary-based operation lookup to keep the code organized and scalable.

## ✨ Features

* Performs addition, subtraction, multiplication, and division
* Supports continuous calculations using previous results
* Allows users to start a new calculation without restarting the program
* Provides an option to exit the application
* Uses a Python dictionary to map operators to functions
* Handles division-by-zero errors gracefully
* Validates user input to prevent invalid operations

## ⚙️ How to Run

Clone this repository or copy the project files

Ensure the following file is in the same folder:

```text
art.py   # Contains the calculator ASCII 
```

Run the script using:

```bash
python calculator.py
```
